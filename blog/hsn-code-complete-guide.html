Quick Summary: HSN (Harmonized System of Nomenclature) is a globally accepted product classification system. Mandatory on GST invoices based on turnover. Learn how to find and use HSN codes.

What is HSN Code?

HSN (Harmonized System of Nomenclature) is a 6-digit international product classification system developed by the World Customs Organization (WCO). India uses 8-digit HSN codes for better classification.

HSN codes are used to classify goods under GST. Each product has a unique HSN code that determines its GST rate.

Example:

HSN Code 8471 = Computers & laptops

GST Rate: 18%

HSN Code Structure

Digits Level Description Example (Laptop)
2-digit Chapter Broad category of products 84 = Machinery
4-digit Heading Sub-category within chapter 8471 = Computers
6-digit Sub-heading Specific product type 847130 = Portable computers
8-digit Tariff Item India-specific classification 84713020 = Laptops
‚úÖ

International Standard: First 6 digits are same worldwide. India adds 2 more digits for detailed classification.

Why is HSN Code Mandatory?

On Invoices

Required on tax invoices based on turnover:

  • Turnover > ‚Çπ5 crore: 6-digit HSN mandatory
  • Turnover ‚Çπ1.5-5 crore: 4-digit HSN mandatory
  • Turnover < ‚Çπ1.5 crore: 4-digit HSN mandatory (from April 2021)

In GST Returns

HSN/SAC codes must be reported in GSTR-1. Helps government track supply patterns and tax collection.

Determines GST Rate

Each HSN code has a specific GST rate (0%, 5%, 12%, 18%, or 28%). Using wrong HSN = wrong tax rate.

HSN vs SAC Codes

For: Goods

Example: 8471 = Computers

Used for physical products, materials, items

SAC Codes

For: Services

Example: 998314 = IT Services

Used for intangible services, consulting, repairs

Remember: Goods = HSN Code | Services = SAC Code (Services Accounting Code)

Penalties for Wrong HSN Code

Consequences of Wrong HSN:

  • Wrong Tax Rate: May result in under-collection or over-collection of GST
  • ITC Denial: Buyer may face ITC rejection if HSN doesn't match product
  • Notices from Department: Tax authorities may issue notices for clarification
  • Penalty: Up to ‚Çπ25,000 for incorrect invoice details
  • Interest: 18% per annum on short-paid tax

‚úÖ How to Correct:

If you discover wrong HSN code on issued invoices, issue a credit/debit note with correct HSN. Inform buyer and correct in next return.

Frequently Asked Questions

Q: Do I need HSN code if turnover is less than ‚Çπ1.5 crore?

A: Yes. Since April 2021, 4-digit HSN is mandatory for all registered taxpayers on B2B invoices, regardless of turnover.

Q: Can one product have multiple HSN codes?

A: No. Each product has one specific HSN code. However, similar products may have different codes based on material, use, or specifications.

Q: Where do I find HSN code for new products?

A: Use our HSN Finder tool with 21,000+ codes, check GST portal, or consult a CA. For entirely new products, refer to customs tariff classification.
