Quick Summary

HSN (Harmonized System of Nomenclature) is a globally standardized system for classifying goods. Under GST, you must mention HSN codes on invoices. Wrong codes = penalties. This guide covers everything from finding your code to avoiding common mistakes.

What is HSN Code?

HSN stands for Harmonized System of Nomenclature. It's a 6-digit code developed by the World Customs Organization (WCO) to classify products uniformly across the world.

Think of it as a universal product ID. Whether you're in India, USA, or Japan ‚Äî a laptop has the same HSN code (8471). This makes international trade easier and helps governments track imports, exports, and tax collection.

India adopted HSN codes under GST to:

  • Classify goods systematically for tax purposes
  • Determine the correct GST rate for each product
  • Track movement of goods across the country
  • Reduce tax disputes and classification errors
  • Align with global trade standards

Understanding HSN Code Structure

HSN codes are hierarchical ‚Äî they go from broad categories to very specific products:

üî¢

HSN Code Breakdown Example: Mobile Phone (8517)

Chapter (2 digits): 85 ‚Äî Electrical machinery and equipment
Heading (4 digits): 8517 ‚Äî Telephone sets, other apparatus for transmission
Subheading (6 digits): 851712 ‚Äî Telephones for cellular networks (smartphones)
Tariff Item (8 digits): 85171290 ‚Äî Other mobile phones (India-specific classification)

Structure: Chapter ‚Üí Heading ‚Üí Subheading ‚Üí Tariff Item

Digit Level What It Means Example
2 digits Chapter ‚Äî Broad category 85 = Electrical machinery
4 digits Heading ‚Äî Product group 8517 = Telephone equipment
6 digits Subheading ‚Äî Specific product type 851712 = Mobile phones
8 digits Tariff Item ‚Äî India-specific detail 85171290 = Other smartphones

HSN vs SAC ‚Äî What's the Difference?

HSN is for goods, SAC is for services.

HSN (Harmonized System of Nomenclature)

Used for: Physical goods ‚Äî products, materials, merchandise
Example: Rice (1006), Cement (2523), Laptop (8471)
Digits: 2, 4, 6, or 8 digits depending on turnover

SAC (Services Accounting Code)

Used for: Services ‚Äî consulting, software, transport, etc.
Example: IT Services (9983), Legal Services (9982), Restaurant (996331)
Digits: Always 6 digits for services

Important Note

If your business sells both goods and services, you'll use both HSN and SAC codes on different invoices. For example: A restaurant uses SAC 996331 for food service, but HSN 2106 for packaged food items sold separately.

5 Common HSN Code Mistakes to Avoid

1

Using Wrong Number of Digits

What happens: Turnover is ‚Çπ8 crore, but using only 4-digit HSN instead of mandatory 6-digit.

Why it's wrong: HSN digit requirement is based on turnover. Using fewer digits than required violates GST rules.

üí∞ Penalty: ‚Çπ10,000 per invoice + possible audit

‚úÖ How to Fix

Check your previous year's turnover. Use 4 digits if under ‚Çπ5 Cr, 6 digits if ‚Çπ5-500 Cr, 8 digits if above ‚Çπ500 Cr or exporting.

2

Completely Missing HSN Code on Invoice

What happens: Invoice doesn't mention any HSN code at all ‚Äî just product name and price.

Why it's wrong: HSN is mandatory on all B2B invoices. Missing it makes invoice non-compliant.

üí∞ Penalty: ‚Çπ10,000 minimum + ITC disallowed for buyer

‚úÖ How to Fix

Update your invoice template to include HSN code field. Most accounting software (Tally, Zoho) auto-fills HSN once you add product master data.

3

Using Generic/Incorrect HSN Code

What happens: All products labeled with same generic HSN like 9999 or random code that doesn't match actual product.

Why it's wrong: Each product has a specific HSN. Using wrong code can lead to wrong GST rate application and tax disputes.

‚ö†Ô∏è Risk: Tax demand notice + interest + ‚Çπ10,000 penalty

‚úÖ How to Fix

Research and assign correct HSN for each product category you sell. Use our HSN finder tool or check government database. Don't guess!

4

Confusing HSN with SAC (or Vice Versa)

What happens: Using HSN code on service invoice, or SAC code on goods invoice.

Why it's wrong: HSN is only for goods, SAC is only for services. Mixing them creates return filing errors.

‚ö†Ô∏è Risk: GSTR-1 rejection + manual correction needed

‚úÖ How to Fix

Remember: HSN = Goods (products you can touch), SAC = Services (work/expertise). Check if your invoice is for goods or services before coding.

5

Not Updating HSN When GST Rates Change

What happens: Government changes GST rate for a product, but business continues using old HSN/rate combination.

Why it's wrong: HSN itself doesn't change, but the GST rate linked to it can change. Using outdated rates = wrong tax collection.

üí∞ Penalty: Tax shortfall recovery + interest + ‚Çπ10,000

‚úÖ How to Fix

Subscribe to GST Council notifications. Check cbic-gst.gov.in regularly for rate changes. Update your accounting software when rates change.

Special HSN Scenarios

‚ùì

What if my product has multiple uses?

Use HSN based on primary use/function. Example: A phone with camera ‚Üí HSN 8517 (phone), not 8525 (camera). If genuinely ambiguous, consult CA or apply for Advance Ruling from GST Authority.

‚ùì

Can I use the same HSN as my competitor?

Only if you sell the same product. Don't blindly copy. HSN must accurately reflect your actual product. Misclassification to claim lower tax rate = tax evasion.

‚ùì

What about imported goods?

Imports must use 8-digit HSN regardless of turnover. Check your Bill of Entry from Customs ‚Äî it will have the 8-digit code. Use the same in your GST invoices.

‚ùì

Do I need HSN for B2C (retail) invoices?

Yes, if your turnover requires it. B2B and B2C both need HSN based on your turnover threshold. Only exception: Bill of Supply (for composition dealers).

How HSN Determines GST Rate

Each HSN code is linked to a specific GST rate by the GST Council. This is called the "HSN-wise GST Rate Schedule."

üìä

How It Works

Step 1: Identify your product's HSN code
Step 2: Check GST rate schedule (available on cbic-gst.gov.in)
Step 3: Apply the rate mentioned for that HSN chapter/heading

Example: HSN 8471 (Laptops) ‚Üí Schedule mentions 18% GST ‚Üí You charge 18%

Important: Some HSN codes have multiple rates based on specifications. Example: Footwear under ‚Çπ1,000 = 5%, above ‚Çπ1,000 = 18%.

Frequently Asked Questions

‚ùì

Is HSN code same across all countries?

First 6 digits are globally uniform (defined by WCO). The 7th and 8th digits are country-specific. So HSN 8471 for laptops is same worldwide, but 84713000 may vary by country.

‚ùì

Do services have HSN codes?

No. Services use SAC (Services Accounting Code), not HSN. SAC is always 6 digits. Example: IT services = SAC 9983.

‚ùì

Can I change HSN code after issuing invoice?

Not easily. You'd need to issue a credit note canceling the wrong invoice and create a fresh invoice with correct HSN. Avoid by double-checking before billing.

‚ùì

Where can I find the official HSN list?

Visit cbic-gst.gov.in ‚Üí Services ‚Üí HSN Lookup or download the complete HSN PDF from CBIC website. Our tool also covers 21,000+ codes with descriptions.

‚ùì

What if I make a custom/unique product?

Find the closest HSN category based on material, function, or industry. If truly unique with no match, consult CA or apply for Advance Ruling from GST Authority for official classification.
