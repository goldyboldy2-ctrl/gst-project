Quick Summary

HSN (Harmonized System of Nomenclature) is a globally accepted product classification system. Mandatory on GST invoices based on turnover. Learn how to find and use HSN codes correctly.

What is HSN Code?

HSN (Harmonized System of Nomenclature) is a 6-digit international product classification system developed by the World Customs Organization (WCO). India uses 8-digit HSN codes for better classification.

HSN codes are used to classify goods under GST. Each product has a unique HSN code that determines its GST rate.

Example

HSN Code 8471 = Computers & laptops

GST Rate: 18%

HSN Code Structure

HSN codes are hierarchical - from broad categories to specific products:

Digits Level Example (Laptop)
2-digit Chapter 84 - Machinery
4-digit Heading 8471 - Computers
6-digit Sub-heading 847130 - Portable computers
8-digit Tariff item 84713010 - Laptops

When is HSN Mandatory?

HSN Code Best Practices

  • Be Specific: Use the most detailed HSN code available for your product
  • Stay Consistent: Use the same HSN code for the same product across all invoices
  • Verify Rates: HSN code determines GST rate - double-check before invoicing
  • Update Regularly: GST Council updates rates - review codes quarterly
  • Document Classification: Keep reasoning for HSN code selection in case of audits
  • Consult CA: If unsure about classification, consult a chartered accountant
