Audit Process

  1. Notice (ADT-01)
  2. Document submission
  3. Auditor examination
  4. Audit report (GSTR-9C)
  5. File annual return (GSTR-9)

Documents Required

  • GSTR returns
  • ITC claims documentation
  • Stock register
  • Ledgers

  • GSTR-2B reconciliation

  • Section 17(5) blocked credits
  • Correct HSN codes

How to Prepare

  • Maintain proper records
  • Reconcile monthly
  • Keep digital backups
  • Hire CA for GSTR-9C
  • Respond to notices promptly
